# PicrossG3

### Introduction

Nous avons comme objectif de créer une application (mini jeu) de jeu *Picross*

### Installation

* `git clone https://github.com/kirozz/PicrossG3.git`
* `cd PicrossG3`
* `gem install bundle` ou `sudo gem install bundle`
* `bundle`
* `rake`
